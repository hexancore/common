## Description

Common module for [Hexancore](https://hexancore.github.io/hexancore).
Module contains basic stuff to use in whole framework and beyond.

## Installation
```bash
# NPM
npm i --save @hexancore/common

# YARN
yarn add @hexancore/common
```
